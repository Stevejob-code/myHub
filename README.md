# MyHub v4 Watchlist Upgrade

อัปเกรดจาก v4 Phase 1 โดยปรับหน้า Watchlist ให้เป็นคลังหนัง/ซีรีส์มากขึ้น

## เพิ่มในเวอร์ชันนี้
- การ์ดหนัง/ซีรีส์แบบมีโปสเตอร์
- เพิ่มข้อมูลแนว, แพลตฟอร์ม, ปี, คะแนน
- กรองตามสถานะ อยากดู / กำลังดู / ดูจบแล้ว
- ค้นหาจากชื่อเรื่อง, แนว, แพลตฟอร์ม, ปี, โน้ต
- หน้าแก้ไขรายการรองรับข้อมูลหนังครบขึ้น

## วิธีใช้
อัปโหลดไฟล์ทั้งหมดขึ้น GitHub repo เดิม แล้ว Cloudflare Pages จะ deploy ให้อัตโนมัติ

หมายเหตุ: ช่องโปสเตอร์ใช้ URL รูปภาพ ถ้าไม่ใส่ ระบบจะแสดงโปสเตอร์สำรองให้

# MyHub v3

Web App ส่วนตัวสำหรับมือถือ: Dashboard, รายรับรายจ่าย, งาน, Watchlist, Notes/Links และ Profile

## สิ่งที่เพิ่มใน v2
- Quick Add แบบ Bottom Sheet
- แก้ไข / ลบข้อมูลทุกโมดูล
- Filter รายรับ/รายจ่าย
- Filter งานทั้งหมด/ค้าง/เสร็จแล้ว
- ค้นหาหนัง/ซีรีส์ และโน้ต/ลิงก์
- UI มือถือเนียนขึ้น

## วิธี deploy
ใช้ Cloudflare Pages:
- Framework preset: None
- Build command: เว้นว่าง
- Build output directory: /

## Firebase
ไฟล์ config อยู่ที่ `src/js/firebase.js`
เปิดใช้งาน:
- Authentication > Email/Password
- Firestore Database

## Firestore Rules
ดูตัวอย่างได้ใน `firestore.rules`


## สิ่งที่เพิ่มใน v3
- PWA ติดตั้งเป็นแอปได้
- Dashboard มีกราฟรายจ่าย 7 วัน
- สรุปหมวดใช้เงินเยอะสุด งานเลยกำหนด Watchlist และโน้ต
- Profile มีสถิติส่วนตัว
- สลับธีมมืด/สว่าง

## Watchlist v4.1
- ลดช่องกรอก: ชื่อเรื่อง, สถานะ, แนว, ปี, คะแนน, ลิงก์โปสเตอร์
- เลือกแพลตฟอร์มจากปุ่มพร้อมไอคอน
- แสดงแท็บสถานะในรายการ
- รองรับดึงโปสเตอร์อัตโนมัติผ่าน TMDB API Key: เปิดไฟล์ `src/js/app.js` แล้วใส่ค่าใน `const TMDB_API_KEY = '';`

## v4.2 Watchlist cleanup
- ลดฟอร์มเหลือ ชื่อเรื่อง, หมวดหมู่, แพลตฟอร์ม, สถานะ
- แพลตฟอร์มเปลี่ยนเป็น dropdown
- หมวดหมู่: หนัง / ซีรีส์ / อนิเมะ / สารคดี
- โปสเตอร์ดึงอัตโนมัติจากชื่อเรื่องผ่าน TMDB เมื่อใส่ API Key ใน `src/js/app.js`


## v4.7 Premium Watchlist
- ปรับปุ่มสถานะในรายการเป็น segmented control แบบ premium
- ลดระยะห่างปุ่ม กำลังดู / ดูจบแล้ว ให้ชิดพอดี
- ปรับปุ่มแก้ไข/ลบให้ compact ขึ้น


## v5.1 Tasks UX
- รีดีไซน์หน้า Tasks ให้ใช้ง่ายขึ้น
- Quick Add เป็นจุดหลัก
- เพิ่มโซน “งานวันนี้”
- Card งานใหม่ ปุ่มทำเสร็จใหญ่ กดง่าย
- ตัด Dashboard/Stats ที่ทำให้หน้าใช้งานงงออก

## v5.8 Subtask UI Premium
- เพิ่มงานย่อยได้หลายรายการจากช่อง Quick Add
- งานย่อยแสดงเป็น checklist พร้อม progress bar ในการ์ด
- แก้ไขงานย่อยได้จากหน้าแก้ไขงาน
- ติ๊กงานย่อยเสร็จ/ไม่เสร็จได้ในรายการ
