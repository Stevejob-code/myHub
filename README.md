# MyHub v7 Money Premium

อัปเกรดหน้า Money ให้เป็นแดชบอร์ดการเงินจริงจังขึ้น พร้อม UI มือถือแบบ premium

## เพิ่ม/ปรับปรุง
- Money Control สรุปคงเหลือ รายรับ รายจ่าย และยอดใช้วันนี้
- Quick Add บันทึกเงินเร็ว พร้อมปุ่มจำนวนเงินลัด
- เลือกประเภทรายรับ/รายจ่ายแบบ segmented button
- หมวดหมู่แบบ chip พร้อมไอคอน
- Money Pulse กราฟรายจ่าย 7 วันล่าสุด
- Top Categories วิเคราะห์หมวดที่ใช้เงินเยอะ
- รายการเงินแบบ card ใหม่ พร้อมไอคอนหมวด สีรายรับ/รายจ่าย
- Filter + Search รายการเงิน

## Deploy
อัปโหลดไฟล์ทั้งหมดขึ้น GitHub repo เดิม แล้ว Cloudflare Pages จะ deploy อัตโนมัติ

## Firebase
ใช้ Firebase config เดิมของโปรเจค myhub-jobz
